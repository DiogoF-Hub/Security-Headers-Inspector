Restricted Page

This page cannot be scanned by browser extensions.

Chromium has a hardcoded list of protected domains built into its source code. This applies to all Chromium-based browsers (Chrome, Brave, Edge, Opera, Vivaldi, etc.). Extensions are blocked before any request is even made — this is a browser-level security boundary, not a server-side decision.

Both the webRequest API (which captures headers during navigation) and fetch requests from extensions are rejected by Chrome on these domains.

This is a browser-level restriction — no extension can bypass it.

External scanners like SecurityHeaders.com work because they make the request from their own servers, outside of the browser sandbox.
